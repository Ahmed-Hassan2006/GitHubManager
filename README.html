ü§î What is GitHub anyway?

Imagine you are writing a book or coding a program. You save a file today. Tomorrow, you make edits. The next day, you delete a paragraph. Suddenly, you realize that the paragraph you deleted yesterday was actually important today!

This highlights the importance of GitHub in uploading versions, whether experimental or stable.

GitHub is like a magical world for your files. It is a cloud storage service (like Google Drive or Dropbox), but with superpowers:

  • It saves every version of your file (providing a log of your edits and actions, known as History).
  • It allows you to share your work with the world (Open Source).
  • It allows people to report problems related to your work (Issues).
  • It allows you to package your work for download (Releases).

In GitHub, a project folder is called a Repository (or "Repo" for short).

üîë Step 1: Setup

Before the program can communicate with GitHub or use the new AI features, it needs permission. You only need to do this once.

1. The GitHub Token (Required)

This is the master key that allows the app to manage your files.

  1. Click this link: Generate GitHub Token (You will need to sign in).
  2. Note Name: Type "GitHub Manager App" or any name you choose.
  3. Expiration: Choose "No expiration" (so you don't have to repeat this), or select any duration you prefer.
  4. Select Scopes: If you used the link above, the correct boxes are already checked for you! (repo, user, delete_repo).
  5. Scroll to the bottom and click "Generate token".
  6. Copy it NOW! You will see a long string starting with ghp_. Copy it immediately.

2. Google AI Key (Optional)

If you want to use the new AI Chat (Assistant) to help you code or answer questions, you need this free key.

  1. Go here: Get Google Gemini Key
  2. Click on "Create API Key".
  3. Copy the key that appears (it starts with AIza).

3. Entering the Keys

Open GitHub Manager. The login menu might appear automatically. If not, press Ctrl + E. Paste your GitHub Token in the box. If you have a Google Key, paste it in the box inside the Settings window within the AI Chat window. Click Save.

üñ•Ô∏è The Interface & Features

The program is divided into two main areas: The Dashboard and the Repository Tab.

1. The Dashboard (Main Window)

This is your main headquarters. Here you can see a list of all your projects.

  • My Repos: Updates the list to show your projects.
  • Advanced Search: Allows you to find other repositories or users on GitHub.
  • Create Repo: Starts a brand new blank project on GitHub.
  • Notifications (New): Shows you if someone starred your repo or commented on your issues.
  • Dark Mode (New): Changes colors to be comfortable for your eyes at night, or to help visually impaired users work in a comfortable environment.

2. The Repository Tab (Inside a Project)

When you press Enter on a repo, a new tab opens.

Files & Folders

You will see a list of files. You can browse them exactly like in Windows Explorer. Press Enter to open files or enter folders.

üë• Team Manager (New)

Want to work with a friend? You can now add "Collaborators" to your project. You can give them read-only access, or permission to edit (Push) your code.

üåø Branches

You will see a dropdown menu labeled "Branches". Imagine a Branch as a parallel universe. The main branch is the real world. You can create a new branch to test ideas safely.

üêõ Issues (Problems & Tasks)

It is a task list or forum for your project.
In the App: You can read discussions, reply to them, close solved problems, or create new issues.

Releases (The Finished Product)

View all project versions and download the attached files (Assets) like ZIP or EXE files.

‚ùì FAQ & Troubleshooting

The app says "Repo Not Found" when I try to open a link.

Ensure the link is correct. It should look like owner/repo-name. Also, make sure your Token has permission to access private repositories.

I see "Binary File Error".

This happens if you try to open an image, PDF, or executable (EXE) file inside the app's internal text reader. The app protects you from freezing. Please use Ctrl + D to download and view these files on your computer instead.

Does it update automatically?

Yes! Every time you open the app, it checks a secure file. If a new version (like v1.2.0) is released, it will ask if you want to update and will handle the installation for you.

Can I manage Private Repos?

Yes, as long as you checked the "repo" box when creating your Token in Step 1.
